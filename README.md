# Robottio

**Cyberpunk-styled web interface для квалификации задач и подбора специализированных агентов**

Robottio - это React-приложение с уникальным UI в стиле Matrix/cyberpunk, которое служит интерфейсом между пользователем и системой автоматизации задач через Telegram-бота.

---

## Особенности

- **Cyberpunk UI** с эффектом глитча
- **Matrix-style фон** с анимированным "дождём" символов
- **Анимированный логотип** с синхронизированными глитч-эффектами
- **Интеграция с Telegram** для обработки задач
- **Современный стек**: React 19.2, Vite

## Архитектура

### Структура проекта

```
src/
├── App.jsx                    # Главный компонент с оркестрацией глитч-эффектов
├── main.jsx                   # Точка входа
├── index.css                  # Глобальные стили
└── components/
    ├── QualifierAgent.jsx     # Основной UI: текстовое поле и кнопка
    ├── MatrixBackground.jsx   # Анимированный Matrix-фон
    └── AnimatedLogo.jsx       # Логотип с анимациями
```

### Ключевые компоненты

**QualifierAgent** - главный интерактивный компонент с двумя режимами:
- **Обычный режим**: описание задачи для квалификации
- **Глитч-режим**: переход к Telegram-боту через активную ссылку

**Система глитчей**:
- Синхронизированные эффекты на всех компонентах
- Срабатывает случайно каждые 15-20 секунд
- Длительность: 3-5 "рывков" по 1 секунде

## Интеграция с n8n

Проект является частью экосистемы автоматизации на базе n8n. Telegram-бот запускает воркфлоу для обработки и распределения задач между специализированными агентами.

## Разработка

### Соглашения о коде

- Компоненты используют React Hooks (`useState`, `useEffect`, `useRef`)
- CSS-классы для глитч-анимаций: `.glitching`, `.border-glitching`, `.glitching-text`
- Таймеры управляются через `setTimeout` с очисткой в `useEffect` return

### Важные детали

- Placeholder в textarea скремблируется во время глитча
- Скремблирование останавливается при фокусе на поле ввода
- Кнопка становится активной ссылкой только в глитч-режиме
